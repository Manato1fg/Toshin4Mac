東進受講.appのインストールの方法

1. ここをクリックしてダウンロードして開きます。

2. 開いたら東進受講.appをApplicationフォルダに入れます。

3. 多分そのままだと開かないので、東進受講.appをクリックして、Finderの歯車のボタンをクリックして、Optionキーを押しながら開くを選択すると開きます。

※初回起動時は必要ファイルのダウンロード(約40MB)をするため、時間がかかります。
